# Audio files

Drop recorded .mp3 files here, named exactly as listed (case-sensitive).
Each name maps to one button already wired into the site. A missing file
shows "Audio coming soon" and stays disabled, so you can add them one at a time.

## Your voice — the "Listen" buttons
- maqamat.mp3   -> opening button under the title
- ataba.mp3     -> First Station header (pronunciation and meaning)
- amanah.mp3    -> Second Station header
- marifa.mp3    -> Third Station header
- tarjama.mp3   -> Fourth Station header
- darwish.mp3   -> closing reading (Darwish poem)

## Your friend's voice — the "Entry Connection" buttons
- ataba_connection.mp3    -> end of First Station
- amanah_connection.mp3   -> end of Second Station
- marifa_connection.mp3   -> end of Third Station
- tarjama_connection.mp3  -> end of Fourth Station
